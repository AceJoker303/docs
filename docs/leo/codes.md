# Radio Callsigns, Codes, and Alphabet

## Callsign Definitions

`{department}{rank/unit type}`-`{2 digit unit number}`
Departments:
- 1 = BCSO
- 2 = LSSO
- 3 = LSPD
- 4 = SSPD
- 5 = SAHP

Rank/Unit Type:
- A = Air Unit
- B = Standard
- C = Captain/Command
- S = Sargent
- L = LT
- K = K9
- T = Two/Dual Units
- N = Probationary Deputy
- U = Standard
- V = Corporal
- M = Marine
